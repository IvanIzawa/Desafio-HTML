Título do Glossário

Primeiro
HTML não é uma linguagem de programação, é uma linguagem de marcação
Segundo
iniciamos um documento html é com "<!DOCTYPE html> " depois adicionamos a tag <html>
Terceiro
dentro do documento html temos a tag "head" onde temos a configuração da pagina e o título da guia,
alem da head temos a tag "body" onde temos o conteudo da pagina.
Quarto
para estilizar o documento podemos usar uma tag "link" para ligar um documento em css
e assim deixar sua pagina mais agradavel.
